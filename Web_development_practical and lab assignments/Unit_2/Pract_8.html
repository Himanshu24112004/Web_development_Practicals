class selector

Syntax : .classname

Class selectors target elements with a specific class attribute and are denoted by a period (.) followed by the class name.
They're reusable and can be applied to multiple elements on the same page.

id selector

Syntax : #idname

ID selectors target a single, unique element using the hash (#) symbol followed by the ID name.
Each ID must be unique within a page.

group selector

Syntax : selector1,selector2,selector3 { properties value... }

Group selectors allow you to apply the same styles to multiple different selectors by separating them with commas.
This reduces code duplication and improves maintainability. â€‹

universal selector

Syntax : * { property: value; }

Universal selectors target all elements on a page or all elements within a specified parent, using the asterisk (*) symbol.
They're commonly used for CSS resets and global styling

descendant selector

Syntax : ancestor descendant { property: value; }

Descendant selectors target elements that are nested inside other elements, using a space character to separate the ancestor and descendant.
They select elements at any nesting level, not just direct children
