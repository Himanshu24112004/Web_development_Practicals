             (Ordered List) Tag

Represents a list of items with a specific order or sequence.

List items are automatically numbered by the browser.

Used when the order of the items matters (e.g., instructions, rankings, steps in a process).

Each item within the list is defined using the 
          1.  (list item) tag.


               (Unordered List) Tag

Represents a list of items without any specific order.

List items are usually displayed with bullet points.

Used when the order of items does not matter (e.g., features, shopping lists, ingredients).
